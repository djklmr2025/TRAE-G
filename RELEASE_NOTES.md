# TRAE-G Release Notes & Instructions

## Manual Release Required
The build artifact `TraeG-Arkaios-v1.0.0.zip` is too large to be hosted directly in this git repository. Please create a release on GitHub manually.

## How to Create the Release

1.  **Navigate to**: [https://github.com/djklmr2025/TRAE-G/releases/new](https://github.com/djklmr2025/TRAE-G/releases/new)
2.  **Tag version**: `v1.0.0`
3.  **Release title**: `TRAE-G v1.0.0`
4.  **Description**:
    ```markdown
    Initial release of TRAE-G (Custom Trae Build).
    Includes Arkaios integration and Nativefier build.
    ```
5.  **Upload Binary**:
    - Upload the file: `c:\arkaios\TRAE-G\TraeG-Arkaios-v1.0.0.zip`
6.  **Publish**.
